# Resetting Password

To reset your password, follow these steps:

1. On login screen, click the **I have forgotten my password** link.![Resetting Password](ResettingPassword1.png)
	The following screen appears:![ResettingPassword](ResettingPassword3.png)
2. Enter you email ID and verify via reCAPTCHA.
3. Click the **Send Password Reset Instructions** button.

If the email entered is valid and exists on the system, instructions to set a new password will be sent over email.
